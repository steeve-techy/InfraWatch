# InfraWatch

Realtime infrastructure monitoring dashboard built using React, Node.js, Socket.IO, and PostgreSQL.

---

## 🚀 Features

- Live CPU, RAM, and Disk monitoring
- Realtime telemetry updates using Socket.IO
- PostgreSQL metric storage
- Historical analytics charts
- Realtime alert system
- Responsive modern dashboard UI
- Optimized telemetry rendering

---

## 🛠 Tech Stack

### Frontend
- React
- Recharts
- Socket.IO Client

### Backend
- Node.js
- Express
- Socket.IO

### Database
- PostgreSQL

---

## 📊 Architecture

```txt
React Dashboard
       ↓
Socket.IO + REST API
       ↓
Node.js Backend
       ↓
PostgreSQL Database
```

## ⚡ Installation

### Clone Repository

```bash
git clone <your-repo-url>
```

### Frontend

```bash
cd client
npm install
npm run dev
```

### Backend

```bash
cd server
npm install
npm start
```
